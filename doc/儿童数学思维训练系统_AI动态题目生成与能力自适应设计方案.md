# 儿童数学思维训练系统：AI动态题目生成与能力自适应设计方案

## 一、设计理念调整

传统固定题库模式：

    题库JSON
    ↓
    随机抽题
    ↓
    孩子答题

存在问题：

-   题目容易重复
-   无法根据孩子能力调整
-   无法发现薄弱能力
-   不能形成长期训练闭环

建议升级为：

> AI数学思维教练系统

整体流程：

    孩子表现数据
    ↓
    AI能力分析
    ↓
    生成每日训练计划
    ↓
    AI生成个性化题目
    ↓
    孩子训练
    ↓
    更新能力模型

------------------------------------------------------------------------

## 二、固定框架，动态生成题目

不要保存大量固定题目。

保存：

-   数学能力体系
-   游戏类型
-   题目生成模板
-   难度规则
-   评价标准

例如：

``` json
{
  "skill":"number-sense",
  "game":"make-ten",
  "goal":"培养补数能力",
  "generator":{
    "targetNumbers":[10,15,20],
    "numberRange":[5,20]
  }
}
```

AI根据模板生成不同题目：

今天：

> 谁和8组成10？

明天：

> 小兔有8颗星星，还需要几颗成为10颗？

后天：

> 哪两个数字可以组成10？

训练目标一致，但题目体验不同。

------------------------------------------------------------------------

## 三、儿童能力画像

AI生成题目的依据不是随机，而是孩子能力。

保存：

``` json
{
 "numberSense":{
   "score":75,
   "weak":["拆数"]
 },
 "relation":{
   "score":55,
   "weak":["多多少问题"]
 }
}
```

分析指标：

-   正确率
-   答题时间
-   提示次数
-   重试次数
-   错误类型

------------------------------------------------------------------------

## 四、每日训练计划

不要每天随机出题。

应该生成：

    热身：
    简单数感游戏

    核心：
    针对薄弱能力训练

    挑战：
    提升思维能力

    趣味：
    保持兴趣

------------------------------------------------------------------------

## 五、AI生成提示词原则

不要：

    生成10道数学题

应该：

    你是一名儿童数学思维训练专家。

    根据孩子能力画像，
    生成15分钟训练任务。

    要求：

    - 不训练计算速度
    - 强调观察、推理、尝试
    - 阅读量少
    - 支持网页游戏交互
    - 适合7岁儿童

    输出：

    skill
    game
    difficulty
    question
    options
    answer
    hint
    thinking
    encouragement

------------------------------------------------------------------------

## 六、动态难度模型

不要简单使用：

    difficulty:1/2/3

建议：

``` json
{
 "difficulty":{
   "numberSize":2,
   "thinkingSteps":2,
   "informationAmount":1,
   "abstraction":2,
   "operationComplexity":1
 }
}
```

综合判断：

-   数字大小
-   思考步骤
-   信息量
-   抽象程度
-   操作复杂度

------------------------------------------------------------------------

## 七、训练反馈闭环

保存：

### 做题记录

``` json
{
 "skill":"relation",
 "result":"wrong",
 "hintUsed":true,
 "time":45
}
```

### 能力变化

``` json
{
 "numberSense":78,
 "relation":62,
 "pattern":85,
 "space":70
}
```

AI根据变化调整下一次训练。

## 九、产品定位

最终产品不是：

> 在线数学题库

而是：

> 面向6-8岁儿童的AI数学思维训练教练。

核心价值：

-   每天生成不同任务
-   根据孩子调整难度
-   发现思维弱点
-   用游戏培养数学兴趣
